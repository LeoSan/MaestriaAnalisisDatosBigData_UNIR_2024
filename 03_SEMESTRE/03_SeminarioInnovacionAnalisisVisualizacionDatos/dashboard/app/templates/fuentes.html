📊 Fuente Final de Datos (Dataset Consolidado)

El conjunto de datos final utilizado para renderizar este Dashboard es el resultado de la extracción, limpieza e integración de múltiples fuentes gubernamentales. Este proceso se llevó a cabo para obtener un panel consolidado a nivel entidad federativa, permitiendo nutrir los modelos de Machine Learning y las visualizaciones.
